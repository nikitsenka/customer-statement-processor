# About the project

Command line interface for CSP REST API interactions

## Users quickstart

Check it works:
```
./csp-cli
```

### Help and examples

To get details about specific command please use `--help` flag. 

For example:
```
./csp-cli --help
```

## Developers

### build:
```
make
```

### Run:

```
./bin/csp-cli
```
